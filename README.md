# LABA2 ( Мини-оболочка с файловыми командами)

## Установка и запуск

### Запуск Лабы
```bash
# Клонирование репозитория
git clone <https://github.com/BagiCafe/Laba2.git>

# Запуск основной программы
python src/main.py
```
## Функционал ( Описание команд)

### Команда ls ( Просмотр содержимого директории)
```bash
ls -l <path>
```
- path - директория для просмотра (по умолчанию текущая)
- -l - подробный вывод

### Команда cd ( Смена директории)
```bash
cd <path>
```
- path - целевая директория
- Специальные символы:
- ~  - домашняя директория
- ..  - родительская директория

### Команда cat ( Вывод содержимого файла)
```bash
cat <file>
```
- file - файл для вывода

### Команда cp ( Копирование файлов и директорий)
```bash
cp -r <source> <purpose>
```
- source - копируемый файл/директория
- purpose - путь назначения
- -r - рекурсивное копирование директорий

### Команда mv ( Перемещение и переименование файлов и директорий)
```bash
mv <source> <purpose>
```
- source - копируемый файл/директория
- purpose - целевой путь

### Команда rm ( Удаление файлов и директорий)
```bash
rm -r <target>
``` 
- target - удаляемый файл/директория
- -r - рекурсивное удаление директорий
- Запрашивает подтверждение для удаления директорий

### Команда zip ( Создание архива)
```bash
zip <catalog> <archive.zip>
```
- catalog - целевые пути (что архивируем)
- archive.zip - путь назначения (куда сохранится архив)

### Команда tar ( Создание архива)
```bash
tar <catalog> <archive.tar.gz>
```
- catalog - целевые пути (что архивируем)
- archive.tar.gz - путь назначения (куда сохранится архив)

### Команда unzip ( Распаковка архива)
```bash
zip <archive.zip>
```
- archive.zip - целевой путь (распаковываемый архив)

### Команда untar ( Распаковка архива)
```bash
tar <archive.tar.gz>
```
- archive.tar.gz - целевой путь (распаковываемый архив)

### Команда grep ( Поиск в файлах)
```bash
grep [flag] <pattern> <path>
```
- pattern - регулярное выражение( что ищем)
- path - целевые пути( где ищем)
- [flag] - Опции:
- -r - рекурсивный поиск в подкаталогах
- -i - поиск без учета регистра

### Логирование
Все команды и их результаты логируются в файл shell.log в формате:
```bash
[2025-10-30 15:42:06] cd - SUCCESS
[2025-10-30 15:42:09] -l - ERROR: Unknown command: -l
```

### Обработка ошибок
Все команды включают обработку ошибок для следующих случаев:
- Несуществующие файлы и директории
- Неправильные аргументы команд
- Ошибки ввода-вывода
- Ошибки прав доступа

**А также для rm - Попытки удаления корневых или родительских каталогов**
